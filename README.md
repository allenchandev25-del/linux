MOVING AROUND
pwd - Shows exactly where you are right now.
ls - Lists all the files in your current folder.
cd - Moves you into a different folder.

MAKING AND DELETING
mkdir - Creates a new folder.
touch - Creates a new blank file.
rm - Deletes a file forever (be careful).

ORGANIZING
cp - Copies a file.
mv - Moves a file to a new place (or renames it).

READING FILES
cat - Dumps the whole text of a file onto the screen.
less - Opens a file so you can scroll through it.

SYSTEM CONTROL
sudo - Runs a command with Administrator power (requires password).
clear - Wipes the screen clean so you have a fresh start.
history - Shows a list of everything you just typed.
man - Shows the instruction manual for a command.
exit - Closes the terminal window.
